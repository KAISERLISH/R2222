二代R2的机构共有7个电机：
AK60-6   *1
DJ2006   *2
DJ3508   *1
DS3520C  *1
DS4255EA *2
以及四个气缸：
左右挡板各一个，限制球的杆一个，爪子一个

控制流程：
为了防止车身超过限制长度，挡板和爪子不能同时在车外
1 取球：挡板打开，辊筒放下，限制杆伸出，辊筒和后面摩擦带上的电机以指定速度旋转吸球，取到球后辊筒停止旋转
2 识别：若为紫球，则下一个摩擦带旋转，限制杆收缩，并继续1。若为红/蓝球，则爪子放下并抓取。
3 放球：挡板收回，蓝球，则爪子放下并抓取，辊筒停止旋转，辊筒抬升，3508抬升，并且2006旋转到指定位置，到达框前爪子松开；2006转回去，3508降下
